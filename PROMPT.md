# UTTC Website Optimization Project

Your job is to turn this site into a conversion-optimized, data-driven sales funnel with a built-in admin intelligence layer.

Core objective:
Build a website that is measurably better at converting visitors, learning from visitor behavior, and evolving itself through an admin panel that monitors performance and recommends changes.

Primary outcomes:
1. Increase click-through and conversion rates.
2. Make the copy maximally persuasive, clear, specific, and action-driven.
3. Make the UX easier to understand, easier to click, and lower friction at every step.
4. Instrument the entire funnel so the company can understand how users behave.
5. Build an admin panel that shows the most important funnel metrics in real time or near real time.
6. Add an AI recommendation layer in the admin panel that reviews the metrics and suggests concrete website changes as data updates.
7. Keep the implementation maintainable inside the existing codebase.

Important product principle:
Optimize for business outcomes, not vanity. Every change must support one of these:
- more clicks
- more conversions
- better user understanding
- better experimentation
- better retention or re-engagement
- lower friction or higher trust

Non-negotiable behavior:
- Do not treat the site as a static design piece.
- Do not rely on vague copy.
- Do not add UI that looks smart but does not help the funnel.
- Do not make changes without measurable purpose.
- Do not ship generic AI suggestions. Recommendations must be tied to actual metrics, segments, and page elements.
